# redis 简介

## 特性

1. 速度块
2. 持久化
3. 数据结构
4. 多语言客户端
5. 功能丰富
6. 简单
7. 复制
8. 高可用分布式

## 典型应用场景

1. 缓存系统
2. 排行榜
3. 计数器
4. 社交网络
5. 消息队列系统
6. 实时系统

## 可执行文件说明

1. redis-server : Redis 服务器
2. redis-cli: Redis 命令行客户端
3. redis-benchmark: Redis 基本性能测试工具
4. redis-check-aof: AOF 文件修复工具
5. redis-check-dump: RDB 文件检查工具
6. redis-sentinel: Sentinel 服务器(2.8 以后)

## 启动方法

1. 最简启动： redis-server

```sh
ps -ef|grep redis
netstat -antpl|grep redis # 检查端口
redis-cli -h ip -p port ping # ping命令
```

2. 动态参数启动
3. 配置文件启动

## 客户端返回值

1. 状态回复： > ping [PONG]
2. 错误回复： hget hello field [(error) WRONGTYPE Operation against]
3. 整数回复： incr hello [(integer) 1]
4. 字符串回复： get hello ["world"]
5. 多行字符串回复： mget hello foo [ 1) "world" 2) "bar" ]

## 常用配置

1. daemonize: 是否是守护进程（no|yes)
2. port: Redis 对外端口号
3. logfile: Redis 系统日志
4. dir: Redis 工作目录

## 命令

```sh
# 查看配置文件，把默认 注释，空格 都去掉，并重定向到 某个文件。
config cat redis-6381.conf|grep -v "#" | grep -v "^$" > redis-6382
```
